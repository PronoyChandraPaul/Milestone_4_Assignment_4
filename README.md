1.getElementById() – নির্দিষ্ট ID দিয়ে একটি element নির্বাচন করে।


getElementsByClassName() – class অনুযায়ী একাধিক element নির্বাচন করে।querySelector() – প্রথম ম্যাচ করা CSS selector নির্বাচন করে।

querySelectorAll() – সব ম্যাচ করা CSS selector নির্বাচন করে।

2.প্রথমে document.createElement('tag') দিয়ে এলিমেন্টটি বানাতে হয়, তারপর appendChild() দিয়ে সেটি পেজের নির্দিষ্ট জায়গায় বসাতে হয়।

3.যে উৎপাদনে ক্লিক করা হয় সেই উপাদান থেকে তার প্যারেন্টের দিকে যাওয়াকে Bubbling বলে।

4.অনেকগুলো আলাদা উপাদানে ইভেন্ট লিসেনার না বসিয়ে, তাদের সবার কমন প্যারেন্ট (Parent) এলিমেন্টে একটি মাত্র লিসেনার বসানোকে Delegation বলে। এতে কোড কম লিখতে হয় এবং মেমোরি বাঁচে।

5.preventDefault(): ব্রাউজারের কোনো স্বাভাবিক কাজ (যেমন: লিংকে ক্লিক করলে অন্য পেজে যাওয়া) বন্ধ করে দেয়।

stopPropagation(): ইভেন্টটি যেন ওপরের প্যারেন্ট এলিমেন্টগুলোর কাছে না পৌঁছায় (অর্থাৎ Bubbling হওয়া) বন্ধ করে দেয়।
